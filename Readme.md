# Data Structures and Algorithms (DSA) Practice

Welcome to my DSA learning repository! This project chronicles my journey through mastering Data Structures and Algorithms using the **Take U Forward (TUF) / Striver's A2Z DSA Sheet**.

All solutions are written in **Java** and aim to cover both brute-force and optimal approaches where applicable.

## 🚀 Progress Tracking

Currently, I have covered the fundamental concepts and basic array problems. Here is the breakdown of the topics completed so far:

### 1. Fundamentals (`src/fundamentals/`)
* **Basic Maths** (`BasicMaths.java`)
  * Count Digits
  * Reverse a Number
  * Print Divisors/Factors
  * Prime Number Check
  * GCD / HCF (Euclidean Algorithm)
  * LCM
  * Armstrong Number Check
* **Basic Recursion** (`BasicRecursion.java`)
* **Basic Sorting Algorithms** (`BasicSorting.java`)
* **Advanced Sorting Algorithms** (`AdvanceSorting.java`)
* **Hashing Basics** (`Hashing.java`)

### 2. Arrays (`src/ArrayPrograms/`)
* **Array Problems** (`ArrayProblems.java`)
  * Find Max and Second Max element
  * Check if an array is sorted
  * Remove duplicates from a sorted array
  * Left Rotate an array (by 1 and by K places)
  * Move zeroes to the end
  * Find the Union of two sorted arrays
  * Find the missing number in an array
  * Maximum Consecutive Ones
  * Find the unique element (appearing once in an array where others appear twice)
  * Longest Subarray with sum K
  * Two Sum problem
  * Longest Subarray with Zero Sum
  * Sort an array of 0s, 1s, and 2s (Dutch National Flag Algorithm)
  * Majority Element (Moore's Voting Algorithm)
* **Searching Algorithms** (`Searching.java`)

## 🛠️ How to Refer to this Project

If you are also following the TUF sheet or learning DSA in Java, you might find this repository helpful:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sidmlv43/dsa.git
   cd dsa
   ```
2. **Open in your IDE:** 
   Import the project into IntelliJ IDEA, Eclipse, or your preferred Java IDE.
3. **Navigate the source code:**
   * Go to the `src/` directory.
   * You'll find packages organized intuitively by topic corresponding to the TUF course structure.
   * Open the `.java` files to inspect my implementations. Most files have a `main` method where you can see the functions in action by running the file directly.
4. **Compare approaches:**
   In files like `BasicMaths.java`, you will find different methodologies implemented separately, like `gcdBruteForce` vs `gcdOptimal`.
